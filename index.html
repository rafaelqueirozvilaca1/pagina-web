Introdução ao Front-end

Olá, hoje vou dar falar sobre as linguagens front-end.

O desenvolvedor front-end lida com tudo que o usuário interage, ou seja, os botões disponíveis para clique, a parte visual e textual do site, o design, a navegação, os layouts, para que o usuário navegue pela página sem problemas.

Veja um resumo sobre as três principais linguagens front-end e as características de cada uma:

Linguagens O que faz
HTML Define o significado e a estrutura do conteúdo da web.
CSS Descreve a apresentação de um documento escrito em HTML (cores, tamanhos, etc).
JavaScript Responsável por dar comportamento dinâmico a página.

HTML

Hipertext Markup Language (HTML) é o componente básico da Web, que estabelece a estrutura básica de um Website.

Algumas das suas características:

  • Não é uma linguagem de programação, por não possuir nenhum recurso de interação com o usuário.
  • Sem o HTML, o navegador não conseguiria exibir textos ou carregar outros conteúdos (imagens, vídeos, tabelas, etc).
  • É fácil de aprender, mesmo para quem está iniciando na programação.
  • Estruturado por tags como parágrafos, imagens, tabelas, listas, etc.

CSS

Cascading Style Sheet (CSS), ou em português, Folhas de Estilo em Cascatas.

Representa a parte visual do site, ou seja, fontes de texto (Times New Roman, Arial, Courier New, etc.), cores, etc., dando estilo ao HTML e personalizando o conteúdo que é visível para o usuário.

Outras características:

  • Assim como HTML, CSS também não é uma linguagem de programação.
  • Permite controle do layout de vários HTML a partir de um único arquivo CSS.
  • Criação de designs responsivos, pensando na usabilidade em qualquer dispositivo eletrônico (computadores, smartphones, tablets, etc.).

JavaScript

JavaScript (JS) representa a parte dinâmica do site.

Características:

  • Diferentemente de HTML e CSS, JavaScript é uma linguagem de programação.
  • Muito baseada em orientação a objetos.
  • Não é uma linguagem muito difícil, principalmente para quem já tem noção sobre a lógica de programação.
  • Linguagem bastante versátil, muito utilizada em projetos para desktop e dispositivos móveis.
